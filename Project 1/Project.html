HTML Summary

HTML(Hyper text Markup Language) is used to define our page conetent structure and meaning. You don't use it for styling purposes.Use CSS for that instead.

  • HTML uses "elements" to describe (anotate) content
  • HTML elements typically have an opening tag, content and a closing tag
  • You can also have void(empty) elements like images.
  • You can also configure elemets with attribures

Learn more about html elemtes on the MDN HTML element reference
